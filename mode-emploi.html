MODE D'EMPLOI

Où coller vos descriptions accessibles
Guide complet pour trouver le champ texte alternatif sur chaque réseau social

INTRODUCTION

Le texte alternatif (ou "alt text") permet aux personnes qui en ont besoin d'avoir accès à vos images, avec un lecteur d'écran, ou sans.

Une fois votre description générée avec l'outil TiRacoon, voici exactement où la coller sur chaque plateforme.

1. TWITTER / X

Étapes :

  1. Composez votre tweet et ajoutez votre image
  2. Cliquez sur le bouton "ALT" qui apparaît en bas à gauche de l'image
  3. Une fenêtre s'ouvre : collez votre description (maximum 1000 caractères)
  4. Cliquez sur "Enregistrer" ou "Save"
Astuce : Si vous ne voyez pas le bouton ALT, activez-le dans Paramètres → Accessibilité, affichage et langues → Accessibilité → cochez "Composer des descriptions d'images"
Documentation Twitter/X

2. FACEBOOK

Étapes :

  1. Créez votre publication et ajoutez votre image
  2. Cliquez sur "Modifier" (icône crayon) en haut à droite de l'image
  3. Sélectionnez "Texte alternatif" dans le menu
  4. Supprimez le texte auto-généré par Facebook
  5. Collez votre description
  6. Cliquez sur "Enregistrer"
Important : Facebook génère automatiquement un texte alt avec son IA. Supprimez-le TOUJOURS et remplacez-le par votre propre description qui est beaucoup plus précise !
Documentation Facebook

3. INSTAGRAM

Étapes :

  1. Préparez votre publication avec votre image
  2. Sur l'écran de partage, faites défiler tout en bas
  3. Cliquez sur "Paramètres avancés"
  4. Trouvez la section "Accessibilité"
  5. Cliquez sur "Écrire un texte alternatif"
  6. Supprimez le texte auto-généré
  7. Collez votre description
  8. Validez et publiez normalement
Note : Instagram génère aussi un texte alternatif "automatique" mais sa qualité laisse à désirer. Remplacez-le par le vôtre pour une vraie accessibilité.
Documentation Instagram

4. LINKEDIN

Étapes :

  1. Créez votre publication et ajoutez votre image
  2. Cliquez sur l'icône "Modifier" (crayon) qui apparaît sur l'image
  3. Sélectionnez "Ajouter un texte de remplacement" ou "Texte alt"
  4. Collez votre description
  5. Cliquez sur "Appliquer" puis publiez
Documentation LinkedIn

5. TIKTOK

Pour les publications de photos :

  1. Créez votre publication avec vos photos
  2. Sur l'écran de publication, cliquez sur "Plus d'options" ou "More options"
  3. Sélectionnez "Ajouter un texte alternatif" ou "Add alternative text"
  4. Collez votre description pour chaque photo
  5. Cliquez sur "Terminé" ou "Done"
Limitation pour les vidéos : TikTok ne propose pas de champ texte alternatif pour les vidéos ni pour les vignettes de vidéos.

Solutions alternatives pour les vidéos :

  • Pour les miniatures/vignettes : Incluez votre description au tout début de la légende de votre vidéo, en commençant par "[Description de l'image] : ..."
  • Pour le contenu vidéo : Activez les sous-titres automatiques ET ajoutez une description/transcription complète dans la légende
  • Recommandation : Décrivez toujours les éléments visuels importants à l'oral dans la vidéo elle-même
Documentation TikTok Accessibilité TikTok

6. YOUTUBE

Limitation : YouTube n'a malheureusement pas de champ texte alternatif dédié pour les miniatures de vidéos.

Solution pour les miniatures :

Décrivez votre miniature dans la description de la vidéo, au début, en écrivant "Miniature : [votre description]"

Pour l'accessibilité globale de vos vidéos :

  • Activez les sous-titres automatiques (YouTube Studio → Sous-titres)
  • Ajoutez une transcription complète dans la description
  • Décrivez oralement les éléments visuels importants dans votre vidéo
Documentation YouTube (Sous-titres)

7. BLUESKY

Étapes :

  1. Composez votre post et ajoutez votre image
  2. Cliquez sur le bouton "ALT" en bas à gauche de l'image
  3. Collez votre description
  4. Validez et publiez
Note : Une option peut vous obliger à rédiger un texte alternatif pour une image avant de poster, une autre offre un affichage plus visible des badges des textes alternatifs.
Aller sur Bluesky

8. MASTODON

Étapes :

  1. Composez votre post et ajoutez votre image
  2. Un champ "Décrire l'image" ou "Description" apparaît automatiquement sous l'image
  3. Collez votre description
  4. Publiez
Note : Mastodon prend l'accessibilité très au sérieux et vous avertit même si vous oubliez de remplir le texte alt !
En savoir plus sur Mastodon

9. SUR VOTRE SITE WEB

Dans le code HTML :

<img src="mon-image.jpg" alt="Votre description ici">

Sur WordPress / Wix / Squarespace / Webflow :

  1. Cliquez sur votre image dans l'éditeur
  2. Cherchez le champ "Texte alternatif", "Alt text" ou "Description alternative" dans les paramètres/propriétés de l'image
  3. Collez votre description
  4. Enregistrez
Guide W3C sur les images accessibles

CAS SPÉCIAUX

CAS SPÉCIAL 1 : LES CAPTURES D'ÉCRAN

Règle d'or : Pour une capture d'écran, vous DEVEZ recopier TOUT le texte visible dans votre description !

Utilisez le générateur en mode "Image" et :

  • Dans "Type d'image" : sélectionnez "Capture d'écran"
  • Dans "Description générale" : précisez de quelle plateforme/application il s'agit (Twitter aka X, article de presse, etc.)
  • Dans "Message" : RECOPIEZ TOUT LE TEXTE visible sur la capture
  • Mentionnez les informations de contexte (nom de l'auteur, date, nombre de likes si pertinent)
Exemple de description générée :
"Capture d'écran d'un tweet de @TiRacoon publié le 3 décembre 2024. Message : "Rendez vos images accessibles ! Chaque description compte pour l'inclusion." Une image accompagne le post (description de l'image) 156 likes, 42 retweets."
Astuce : Si la capture contient énormément de texte (plus de 500 caractères), envisagez d'utiliser textup.fr (voir ci-dessous) ou de faire plusieurs publications avec des captures séparées.

CAS SPÉCIAL 2 : LES COMMUNIQUÉS DE PRESSE

Les communiqués de presse contiennent souvent beaucoup de texte. Pour contourner les limites de caractères des réseaux sociaux, lorsque vous les publiez sous forme d'images, utilisez textup.fr !

Méthode en 5 étapes :

  1. Dans le générateur, sélectionnez "Communiqué de presse"
  2. Remplissez les informations :
    • Choisissez si c'est un nouveau communiqué ou si vous avez déjà un lien
    • Titre du communiqué
    • Description de l'image (en-tête, logos, mise en page du document)
    • Collez le texte COMPLET du communiqué dans le champ prévu
  3. Cliquez sur "Copier le texte" puis "Ouvrir textup.fr"
  4. Sur textup.fr :
    • Collez votre texte complet
    • Cliquez sur "Héberger mon texte"
    • Copiez le lien généré (exemple : https://textup.fr/abc123)
  5. Retournez dans le générateur :
    • Collez le lien textup.fr dans le champ prévu
    • Cliquez sur "Générer la description"
    • Copiez et utilisez cette description comme texte alt de votre image de communiqué
Résultat final :
"Document officiel avec en-tête du Ministère intitulé 'Nouvelle loi sur l'accessibilité'. Version accessible disponible sur : https://textup.fr/abc123"

Avantages de textup.fr :

  • Gratuit et sans inscription
  • Pas de limite de caractères
  • Accessible aux lecteurs d'écran
  • Vous pouvez aussi héberger le texte sur votre propre site web si vous préférez
Accéder à textup.fr

POURQUOI C'EST IMPORTANT

Qui bénéficie du texte alt ?

  • Les personnes qui utilisent des lecteurs d'écran
  • Les personnes qui ont besoin de ces descriptions pour différentes raisons
  • Tout le monde quand les images ne se chargent pas (mauvaise connexion)
  • Les moteurs de recherche qui indexent mieux votre contenu

Chiffres à retenir :

  • Plus de 70% des images sur les réseaux sociaux n'ont pas de texte alternatif
  • Un texte alt bien rédigé prend moins de 2 minutes mais fait toute la différence pour des millions de personnes

En résumé : Rendre vos contenus accessibles, c'est inclure tout le monde dans la conversation numérique. Chaque description compte !
